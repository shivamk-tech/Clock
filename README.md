# Aesthetic Analog Clock

A small, responsive analog clock built with **HTML**, **CSS**, and **JavaScript**.  
Features a premium gradient face, smooth hand motion, and customizable styles — perfect as a tiny UI widget, portfolio piece, or learning project.

## Features
- Responsive circular clock (`54vw`) — scales with viewport
- Smooth, realistic hand movement (hour, minute, second)
- Aesthetic gradient face and body background
- Customizable hand styles (rounded tips, glow, counterweight)
- Easy to tweak sizes, colors, and presets
